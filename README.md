# 🏙️ 中国城市生活成本 & 薪资购买力计算器

一个单页 HTML 应用，用于对比中国不同城市的生活成本和薪资购买力。

## ✨ 功能特点

- 📊 **双模式收入计算**
  - **模式A：同岗位锚定** - 输入上海年薪，系统按城市系数自动折算各地薪资
  - **模式B：相同年薪** - 假设远程工作或外派，所有城市拿一样的钱

- 💰 **生活成本分析**
  - 房租成本（可自定义面积和单价）
  - 餐饮成本（买菜做饭 + 外食聚餐）
  - 交通和杂项支出

- 📈 **购买力对比**
  - 当地平均薪资对比
  - 薪资倍数分析
  - 税后收入计算（扣除五险一金+个税）
  - 年结余计算

- 🎯 **覆盖城市**
  - 一线城市：上海、北京、深圳、广州
  - 新一线城市：杭州、苏州、南京、成都、武汉、重庆、长沙、西安
  - 其他城市：昆山、沈阳

## 🚀 使用方法

1. **直接在浏览器打开** `china_cost_cal.html` 即可使用
2. **调整生活成本参数**（以上海为基准）
3. **选择收入模式**并输入年薪
4. **查看对比结果**：
   - 表一：各城市生活成本明细
   - 表二：薪资购买力和结余分析

## 📱 响应式设计

## 🔧 技术特点

- 纯 HTML/CSS/JavaScript，无需后端
- 无框架依赖，加载快速
- 实时计算，即时反馈
- 数据基于 2024 年市场调研

## 📊 数据说明

- **平均薪资**：估算的 2024 年当地社会平均年薪（税前，含社保公积金单位部分）
- **成本系数**：相对于上海的倍数（1.00 = 与上海相同）
- **薪资系数**：相对于上海的薪资水平倍数
